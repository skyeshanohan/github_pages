    Repository Ownership Tracker

    A centralized dashboard for tracking repository ownership across GitHub organizations, pods, verticals, and engineering managers.

    Features
    Searchable Table

    Quickly search and filter repositories by organization, name, pod, vertical, or engineering manager.

    Statistics Dashboard

    View detailed statistics and breakdowns of repository distribution across your organization.

    Responsive Design

    Fully responsive interface that works on desktop, tablet, and mobile devices.

    Auto-Deployment

    Automatically deployed via GitHub Actions whenever you update the data file.

    How It Works
    1. Repository ownership data is stored in data/repositories.json
    2. Each repository entry contains:
      • GitHub organization
      • Repository name
      • Pod assignment
      • Vertical organization
      • Engineering Manager responsible
    3. The site automatically loads and displays this data
    4. Use the search functionality to quickly find repositories
    5. View statistics to understand distribution patterns
    Updating Data

    To update the repository ownership information, simply edit the data/repositories.json file in your repository. After pushing the changes, GitHub Actions will automatically rebuild and redeploy the site.

    Pro Tip

    This site is built with CoreUI, a modern Bootstrap-based admin template, and is deployed via GitHub Pages. The data is stored in a simple JSON format for easy maintenance.
